## 二、技能使用规则

### 强制使用场景
| 场景 | 技能 |
|------|------|
| 开发新功能 | superpowers + planning-with-files |
| 复杂任务(>5步) | planning-with-files |
| 搜索信息 | baidu-search / multi-search-engine |
| 代码审查 | code-review |
| 设计 UI | superdesign / ui-ux-pro-max |
| 复盘错误 | self-reflecting-chain |

### 自动化触发
- 搜索信息 → 自动使用搜索技能
- 复杂任务 → 自动使用 planning-with-files
- 开发功能 → 自动使用 superpowers
