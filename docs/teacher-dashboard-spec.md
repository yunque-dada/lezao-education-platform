## 一、老师工作台概述

### 1.1 功能定位
老师工作台是乐造AI教务平台的核心管理界面，供老师管理学生、班级、作业和查看作品。

### 1.2 核心功能模块

| 模块 | 功能 |
|------|------|
| **首页** | 统计数据概览 |
| **学生管理** | 查看、添加、编辑、删除学生 |
| **班级管理** | 创建、编辑、删除班级，分配学生 |
| **作业管理** | 创建作业、布置作业、查看提交、评分 |
| **作品管理** | 查看学生作品、点评作品 |

## 四、交互流程设计

### 4.1 学生管理流程
```
点击"学生"tab
    ↓
显示学生列表（默认显示全部）
    ↓
[搜索] → 输入关键词 → 实时筛选
[筛选] → 选择班级 → 显示该班级学生
[添加] → 弹窗 → 填写信息 → 提交 → 刷新列表
[编辑] → 弹窗 → 修改信息 → 保存 → 刷新列表
[删除] → 确认弹窗 → 确认 → 删除 → 刷新列表
```

### 4.2 班级管理流程
```
点击"班级"tab
    ↓
显示班级列表
    ↓
[创建] → 弹窗 → 填写班级信息 → 创建 → 刷新列表
[查看] → 显示班级学生列表
[编辑] → 弹窗 → 修改信息 → 保存
[删除] → 确认 → 删除（学生不会被删除）
```

### 4.3 作业管理流程
```
点击"作业"tab
    ↓
显示作业列表
    ↓
[创建] → 弹窗 → 填写作业信息 → 发布 → 刷新列表
[查看提交] → 进入批改页面
[批改] → 查看作品 → 打分 → 写评语 → 提交
```
