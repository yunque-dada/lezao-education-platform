# 乐造AI教育平台 - 项目需求规格

> 更新：2026-03-22

---

## 1. 项目概述

### 1.1 项目背景

乐造AI教育平台是一个面向K12阶段的AI编程教育平台，提供Scratch和Python编程教学环境，支持老师发布课程、布置作业，学生在线学习、提交作品的一体化教学管理系统。

### 1.2 项目目标

- 为老师提供完整的班级管理、课程发布、作业布置、作品审核功能
- 为学生提供趣味编程学习环境，包括Scratch拖拽编程和Python代码编程
- 为管理员提供用户管理和系统配置功能

### 1.3 项目现状

| 项目 | 状态 |
|------|------|
| 前端页面 | ✅ 6个HTML页面已完成 |
| 后端框架 | ✅ Express.js 已搭建 |
| 数据库 | ⚠️ PostgreSQL 配置待完成 |
| 前后端连接 | ❌ 未连接（数据存LocalStorage） |
| 用户系统 | ⚠️ 基础实现，数据混乱 |

## 2. 功能需求

### 2.1 用户角色

| 角色 | 说明 | 权限 |
|------|------|------|
| 管理员 | admin | 系统管理、用户管理 |

| 学生 | student | 学习课程、完成作业、提交作品 |

- [ ] 作品审核（查看/审核学生作品）
- [ ] 课程管理（发布/编辑课程）

#### 2.2.3 学生端

## 3. 技术栈

### 3.1 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | 原生HTML + CSS + JavaScript | 无框架，轻量级 |
| 后端 | Express.js | Node.js Web框架 |
| 数据库 | PostgreSQL | 关系型数据库 |
| 文件存储 | 本地文件系统 | uploads目录 |

### 3.2 核心依赖


## 5. 非功能需求

### 5.1 性能

- 页面加载时间 < 3秒
- API响应时间 < 500ms

### 5.2 安全

- 密码使用BCrypt加密存储
- JWT令牌认证
- 防止SQL注入
- CORS跨域配置

### 5.3 兼容性

- Chrome/Edge/Firefox 最新版
- 响应式布局支持移动端

## 6. 验收标准

### 6.1 功能验收

- [ ] 用户可以登录并根据角色跳转到对应页面

- [ ] 管理员可以管理系统用户

### 6.2 数据验收

- [ ] 数据存储到PostgreSQL数据库

### 6.3 部署验收

- [ ] 数据库连接正常
- [ ] HTTPS访问正常
